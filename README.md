# Assetto Corsa Adaptive Subdivision

This project aims to implement an adaptive subdivision algorithm for use in Assetto Corsa, enhancing visual detail in real-time rendering.

## Features
- Adaptive subdivision based on the distance from the camera.
- Mesh utilities for easy manipulation of 3D models.
- Configuration files for adjusting algorithm parameters.
